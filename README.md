# Smash Racing

A 1v11 oval-track racing game. Press **F** to turn on smash mode and ram CPU cars.

## How to run

Open `index.html` in your browser (Chrome, Edge, or Firefox recommended).

Or run a local server:

```bash
# If you have Python 3
python3 -m http.server 8000
# Then open http://localhost:8000
```

## Controls

| Key | Action |
|-----|--------|
| **Arrow keys** or **WASD** | Drive (Up/W accelerate, Down/S brake, Left/Right steer) |
| **F** | Smash mode: hold to ram CPUs and send them flying |

## Rules

- Oval track, **3 laps**.
- You + **11 CPUs**, 12 cars total.
- Hold **F** and hit a CPU to smash them (they spin out briefly). Without F, you slow down and get a short stun.
- First to complete 3 laps wins. Finishing order decides the podium.

Good luck.
